# Level 7 -> 8

## 목표
비밀번호는 `data.txt`파일에서 단어 `millionth` 뒤에 있다.

## 사용한 명령어
```bash
ssh
ls
cat
grep
```

### 실행 과정
ssh로 bandit 서버에 원격 연결 후에 (비밀번호 입력) 디렉토리 내에 어떤 파일이 있는지를 확인
```bash
ls
```
`data.txt`가 있음

```bash
cat data.txt
```
문자가 끊임없이 출력되어서 터미널 강제 종료

`millionth`가 있는 줄을 찾고 해당 줄을 출력
```bash
grep 'millionth' data.txt
```
비밀번호 출력


### 정리
`grep (찾고 싶은 문자열) (탐색할 파일)`는 해당 문자열이 포함된 줄을 출력한다.

해당 레벨에서 `data.txt`가 방대한 문자열을 담고있다.

`grep`을 통해 찾는 문자열이 있는 줄만 추출이 가능하다.

* 이번 레벨에서는 찾고자 하는 문자열에 따옴표를 붙이지 않아도 정상적으로 작동이 가능하다.
* 그러나 찾고자 하는 문자열이 특수기호로 시작하는 경우에는 옵션으로 해석될 수 있기 때문에 항상 따옴표를 붙이는 것이 좋다.
