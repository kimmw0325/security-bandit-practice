# Level 3 -> 4

## 목표
inhere 디렉토리 내에 있는 숨겨진 파일을 읽기

## 사용한 명령어
```bash
ssh
cd 
ls -a
cat
```

### 실행 과정
ssh로 bandit 서버에 원격 연결 후에 (비밀번호 입력) 디렉토리 내에 어떤 파일이 있는지를 확인
```bash
ls
```

inhere 디렉토리가 있는 것을 확인. inhere 디렉토리로 이동
```bash
cd inhere/
```

inhere 디렉토리 내에 어떤 파일이 있는지를 확인
```bash
ls
```

아무것도 출력되지 않음. 숨김파일까지 모두 확인하는 커맨드 입력
```bash
ls -a
```

`.  ..  ...Hiding-From-You`출력됨
```bash
cat ./...Hiding-From-You
```

비밀번호 출력됨

### 정리
`cd 디렉토리명/`을 하게 되면 해당 디렉토리로 이동한다.

리눅스에서 이름이 `.`으로 시작하는 파일은 숨김 파일이기에 `ls`로는 찾지 못한다.

`ls -a`를 하게 되면 숨겨진 파일(`.`로 시작하는 파일)도 모두 보여주게 된다.

위 문제에서 `ls -a`를 실행해서 나온 `.`와 `..`는 각각 현재 디렉토리, 부모 디렉토리다.
실제로 `cd ../`를 실행해보면 다시 홈 디렉토리로 돌아간다.

파일명이 특수문자로 시작하기 때문에 `./`로 읽어줌
