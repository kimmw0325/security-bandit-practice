# Level 0 -> 1

## 목표
readme 파일을 읽고, 비밀번호를 찾는 문제

## 사용한 명령어
```bash
ssh 
ls 
cat
```

### 해결 과정
ssh로 bandit 서버에 원격 연결한 후 (초기 비밀번호 입력) 디렉토리 내에 어떤 파일이 있는지를 확인
```bash
ls
```
readme 파일이 있는 것을 확인했고, 파일 내용을 읽음
```bash
cat readme
```
비밀번호가 출력됨

### 정리
리눅스를 잘 다루기 위해서는 UI없이 터미널로 디텍토리를 다루는 것에 능숙해야 한다.

ssh로 bandit의 각 레벨 서버에 접속한다.

ls로 디렉토리 내의 파일들을 읽어낸다.

cat으로 파일의 내용을 읽어낸다.
