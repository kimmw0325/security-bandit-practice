# security-bandit-practice
OverTheWire Bandit을 통해 리눅스 명령어 및 시스템 보안 기초를 학습한 기록입니다.

## 프로젝트 목적

보안 분야에 관심을 갖고, 가장 먼저 리눅스 환경과 기본 명령어에 익숙해지는 것을 목표로 진행한 학습 기록입니다.  
OverTheWire Bandit 워게임을 통해 SSH 접속, 파일 탐색, 권한 확인, 숨김 파일 탐색, 문자열 검색 등 시스템 보안의 기초가 되는 개념을 직접 실습했습니다.

## 실습 환경

- macOS Terminal
- SSH
- OverTheWire Bandit
- GitHub README 기록

## 학습 목표

- 리눅스 명령어에 익숙해지기
- 파일과 디렉터리 구조 이해하기
- 파일 권한과 접근 제어 개념 익히기
- 문제 해결 과정을 글로 정리하는 습관 만들기
- 시스템해킹 학습을 위한 기초 체력 쌓기

## 사용한 주요 명령어

- `ssh`
- `ls`
- `cd`
- `cat`
- `file`
- `find`
- `grep`
- `chmod`
